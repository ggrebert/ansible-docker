## Tools

### Git

Download and Install [Git Bash](https://gitforwindows.org/) and configure it to use Unix file format (LF).

Recommendations:

* Squash all your commits before push
* Rebase from `origin/master` before push

### Microsoft VSCode

The recommended `IDE` is [Visual Studio Code](https://code.visualstudio.com/).
